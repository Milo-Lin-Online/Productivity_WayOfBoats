Connect a free Supabase project so everyone shares the same live data and cursors. Paste your Project URL and anon public key below. Setup steps are at the bottom.

📖 One-time setup steps (click)
  1. Go to supabase.com → sign up free → New Project.
  2. Wait ~2 min for it to build.
  3. Left sidebar → SQL Editor → paste & run this:
    create table boats_state (room text primary key, data jsonb);
    alter table boats_state enable row level security;
    create policy "all" on boats_state for all using (true) with check (true);
    alter publication supabase_realtime add table boats_state;
  4. Left sidebar → Project Settings → API.
  5. Copy Project URL → paste above.
  6. Copy the anon / public key → paste above.
  7. Pick a room name, share it + this page's link with your team, and hit Connect!
